# REVF1 — Reverse F1 2026 Tracker

A fan-made web app that tracks the 2026 F1 season using a **reverse scoring system** — 22nd place gets the most points, because regular standings are too mainstream.

## Scoring
| Position | Race Pts | Sprint Pts |
|----------|----------|------------|
| 22nd | 25 | 8 |
| 21st | 18 | 7 |
| 20th | 15 | 6 |
| 19th | 12 | 5 |
| 18th | 10 | 4 |
| 17th | 8 | 3 |
| 16th | 6 | 2 |
| 15th | 4 | 1 |
| 14th | 2 | — |
| 13th | 1 | — |

## Features
- Live driver standings
- Constructor championship
- Race history
- Points progression charts
- Head-to-head driver comparison

## Data
Results are pulled live from a Google Sheet. No manual entry needed on the site.

## Built With
HTML · CSS · JavaScript · Chart.js · Google Sheets
